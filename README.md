# Evidencia 1 – CRUD DAO y Manejo de Excepciones en Java

##  Objetivo
Implementar un sistema de gestión de funcionarios con CRUD completo utilizando el patrón DAO y manejo de excepciones en Java, conectado a una base de datos MariaDB.

---

##  Tecnologías utilizadas
- Java 17+
- Swing (interfaz gráfica)
- MariaDB / MySQL
- JDBC
- Patrón DAO
